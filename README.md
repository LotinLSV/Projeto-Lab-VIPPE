# Documentação do ambiente de labotatório da VIPPE


## Requisitos
 - Criar ambiente que possua:
    - Acesso ao N8N
    - Acesso a um agente que leia arquivos
    - Ambeinte separado por usuário
    - Ambiente com acesso a upload de arquivos

## Realizado

  - Criado backend usand flask para servidor
  - Criado requisições para o HTML usando request
  - Criado layout de Web App PyWebView
  - Integração com N8N
  - Espaço para armazenamento

## Features futuras
  - S/D
    - Conectar a um banco de verdade para escalabilidade
    - CSS para deixar mais organizado 
    - Conectar o agente a uma API KEY usando o .env
